

**robot486**
**🇷🇺 Бот-помощник для изучения русского языка**

Телеграм-бот для изучения русского языка. Поддерживает регистрацию пользователей, настройку уровня владения языком, личный словарь, упражнения на склонение/спряжение и случайные тесты.

---

**Основные функции:**

- /start – запуск бота и регистрация пользователя
- /help – открыть меню с командами
- /level – просмотреть текущий уровень русского языка
- /view_words – просмотреть личный словарь
- /cancel – выйти из текущего режима ввода
- /exercise – упражнение на склонение или спряжение
- /quiz – случайный тест (русский → китайский или китайский → русский)

**Установка и запуск:**

Рекомендуется использовать виртуальное окружение, чтобы не загрязнять глобальный Python.

cd robot
python -m venv .venv
.\.venv\Scripts\Activate.ps1   # Windows PowerShell
# или source .venv/bin/activate # Linux/macOS
python -m pip install -r requirements.txt
python .\main.py

Можно запускать без активации виртуального окружения:

cd robot
.\.venv\Scripts\python.exe .\main.py

---

**Настройка токена:**

Токен бота временно прописан в main.py в переменной TOKEN.

Важно: перед публикацией кода вынесите его в переменные окружения (например, через .env), чтобы избежать утечки.

---

**Модули, за которые отвечает участник C:**

Участник C разработал модули для упражнений и тестов:

- managers/quiz_manager.py – генерация вопросов, проверка ответов, запись результатов в БД
- handlers/grammar.py – логика команды /exercise
- handlers/quiz.py – логика команды /quiz

Результаты тестов сохраняются в таблицу quiz_history базы данных SQLite.

---

**База данных:**

Файл data/bot_database.db создаётся автоматически при первом запуске. Включать его в репозиторий не нужно – таблицы создадутся сами.
